SETTING UP THE PROJECT. 

PRE-REQUIREMENTS:
 -XAMPP
 -NODE
 -CODE EDITOR OF YOUR CHOICE

1. Importing The Database Structure
go to your XAMMP and click on admin panel on the MYSQL tab

on "File to import" Click on schoose File

Import the file inside the "Expensetracker/server/DatabaseStructure" folder

and finally press the "import" button on the very bottom of the page.

2. Starting the XAMPP Database and Starting Up The Project

Go to your XAMPP and turn on your MySQL Database

After, Go to your code editor and create two seperate terminals with the Directory of 
"ExpenseTracker\client\xpence" 
and 
"ExpenseTracker\server"

on both terminals, run the command "npm run dev"

3. Accessing Expence

after, on your "ExpenseTracker\client\xpence" terminal, there will be a Local Host Address and go to your browser and enter in the Address that appears on the terminal.

After that you will be able to use the project
